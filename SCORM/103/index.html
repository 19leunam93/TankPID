Was ist Modelica

Modelica ist eine objektorientierte Modellierungssprache für physikalische Modelle. Sie ist 1997 im Programmiersprachenstandard 1.0 erschienen. Im April 2017 wurde Version 3.4 veröffentlicht. Ein in Modelica mit algebraischen und gewöhnlichen Differenzialgleichungen formuliertes, physikalisches Modell wird von einem Modelica-Translator in ein mathematisches Modell übersetzt und mittels eines Lösungsalgorithmus gelöst.

Vorteile von Modelica

Ein wesentlicher Vorteil von Modelica besteht darin, dass Modelica mit Gleichungen, anstatt mit Zuweisungen arbeitet. Es muss nicht nach gesuchten Variablen aufgelöst werden.
Ein weiterer Vorteil besteht darin, dass Variablen mit Eigenschaften (physikalische Größe, Einheit) versehen werden können. Dies erlaubt die Prüfung von Gleichungen durch die Simulationssoftware.

Hello World

Das erste eigene Programm in einer neuen Programmiersprache ist traditioneller Weise das "Hello World". Dabei wird normalerweise den String "Hello World" auf dem Bildschirm ausgegeben. Da Modelica jedoch eine gleichungsbasierte Programmiersprache ist, macht es keinen Sinn einen String auszugeben. Stattdessen wird unser "Hello World" Programm ein einfaches physikalisches Problem lösen: Ein Pendel
Aus dem Physikunterricht sind die Gleichungen bekannt, welche dieses System beschreiben:
\(m\dot{v}_{x}=-\frac{x}{L}F\)

\(m\dot{v}_{y}=-\frac{y}{L}F-mg\)

\(\dot{x}=v_{x}\)

\(\dot{y}=v_{y}\)

\(x^2+y^2=L^2\)
Die ersten vier Differentialgleichungen zusammen mit der letzten algebraischen Gleichung stellen ein Set an Gleichungen, welche ein Differential-algebraisches Gleichungssystem genannt wird. Mit konventionellem Code, wäre das Lösen eines solchen physikalischen Modells extrem aufwändig. Mit Modelica ist das ganze mit ein paar Zeilen Code gelöst:

class Pendulum
	constant Real PI=3.141592653589793;
	parameter Real m=1, g=9.81, L=0.5;
	Real F;
	output Real x(start=0.5),y(start=0);
	output Real vx,vy;
equation
	m*der(vx)=-(x/L)*F;
	m*der(vy)=-(y/L)*F-m*g;
	der(x)=vx;
	der(y)=vy;
	x^2+y^2=L^2;
end Pendulum;
Um das oben definierte Modell names "Pendulum" zu simulieren (den physikalischen Versuch virtuell auszuführen) muss noch folgendes Kommando gegeben werden:
1
simulate(Pendulum, stopTime=4)
Nun kann das Ergebnis der Simulation betrachtet werden mit:
1
plot(x)
